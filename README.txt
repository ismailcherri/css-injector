CSS Injector
==============
This module adds the ability to inject CSS rules directly into the page bottom to affect the whole website regardless
of the current theme and/or user. Thus, it affects both the active theme and the admin theme.

Installation
===============
- Download the files and place them under DRUPAL_ROOT/modules
- Enable the module through Drupal's backend or drush

Usage
========
- Visit /admin/config/css_injector/css_injector_config
- Add your CSS rules
- Click Save configuration
